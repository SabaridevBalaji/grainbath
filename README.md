#grainbath

I built this cuz i needed a fast way to batch process wide photos for seamless carousels and soak them in film grain without booting up an editing software like lightroom which is quite heavy. This runs entirely local in the browser

i do virtual photography post them on instagram. sometimes i do these seamless carousel posts of cars. Doing them in photoshop can take a VERY long time and can be quite confusing. So in order to fix that problem i created this web tool.
I will be added more and more features as time goes by.

##stuff it does
* batch uploads for slicing multpile wide format images at once
* adjustable film grain intensity slider built on html5 canvas
* auto calculates aspect ratios and slices images for seamless swiping
* generates individual visual previews and download links for every piece
* fully client side
* minimalist ui
